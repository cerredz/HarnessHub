# HarnessIQ Master Prompts

This directory is the repository entry point that contains all of our master plans for HarnessIQ. The bundled master prompt assets live in [`prompts/`](./prompts), `registry.py` loads them into the SDK, and the `harnessiq prompts` CLI surfaces the same catalog for inspection and activation.

## Maintenance Contract

- When you create a new master prompt under `prompts/`, append its title and description to the catalog below in the same change.
- Keep this README aligned with the live prompt metadata in `prompts/*.json`.
- Treat `artifacts/file_index.md` as generator-owned documentation; update it through `python scripts/sync_repo_docs.py` rather than hand-editing it.
